# BLOOD BANK MANAGEMNET SYSTEM

**Simple Blood Bank Management System implemented as a web application**

 **Frameworks - ReactJs (frontend)** <br/>
 **Backend    - NodeJs** <br/>
 **DataBase   - MySQL** <br/>

### Functionality

#### **User**

- Register & Login
- donate blood 
- request blood 

#### **Employee**

- Register & Login
- Update blood stocks
- update user health
- Handle blood request

#### **General**

- Search - based on blood group or location
